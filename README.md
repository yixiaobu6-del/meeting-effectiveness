# 会议效果评估器

> 会议效率与效果智能评估工具，三维评分与优化建议

---

## Features / 功能特点

| 功能 | 说明 |
|------|------|
| 会议录入 | 主题、类型、人数、时长、议题数、决策数完整记录 |
| 效率得分 | 基于每人每议题时间评估会议效率 |
| 产出得分 | 基于决策数与议题数比例评估产出质量 |
| 必要性得分 | 综合人数/时长/决策等因素评估必要性 |
| 综合评估 | 优秀/良好/一般/低效四个等级评定 |
| 雷达图 | 三维度评分可视化展示 |
| 优化建议 | 针对低分维度自动生成改进建议 |
| 历史追踪 | 所有会议记录存档，支持筛选查看 |
| 趋势图 | 会议评分历史变化折线图 |

## Installation / 安装

无需安装，直接在浏览器中打开 `index.html` 即可使用。

```bash
# 克隆仓库
git clone https://github.com/yourusername/meeting-effectiveness-evaluator.git

cd meeting-effectiveness-evaluator
open index.html
```

## Usage / 使用方法

### 基础用法

1. 打开 `index.html`
2. 在「录入会议」Tab 填写会议信息
3. 点击「评估会议」查看综合评分和改进建议
4. 切换到「历史记录」Tab 查看趋势和过往记录

### 会议录入示例

```javascript
// 会议数据结构
{
  id: 1704067200000,
  title: "产品规划会议",
  type: "决策类",          // 决策类/讨论类/汇报类/协调类
  participants: 8,         // 参会人数
  duration: 60,            // 会议时长（分钟）
  agendaCount: 5,          // 议题数量
  decisionCount: 3,        // 决策数量
  date: "2024-01-01"
}
```

### 评分等级说明

| 等级 | 综合得分 | 说明 | 优化建议 |
|------|:--------:|------|----------|
| 优秀 | 80-100 | 高效高产出会议 | 保持现有做法 |
| 良好 | 60-79 | 整体效果不错 | 可微调细节 |
| 一般 | 40-59 | 存在改进空间 | 需针对性优化 |
| 低效 | 0-39 | 效率或产出不足 | 需重构会议方式 |

### 三维度评分逻辑

| 维度 | 计算公式 | 优化方向 |
|------|----------|----------|
| 效率得分 | 100 - (时长/人数/议题数 × 偏差系数) | 减少人数、压缩时长 |
| 产出得分 | (决策数/议题数) × 100 | 提前准备、明确目标 |
| 必要性得分 | 多因素加权综合评估 | 考虑是否需要开会 |
